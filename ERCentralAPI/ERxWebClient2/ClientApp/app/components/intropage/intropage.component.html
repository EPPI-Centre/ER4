Welcome to EPPI-Reviewer Web (beta).

This Beta Application provides an alternative intreface to many common functionalities available in EPPI-Reviewer 4.

It works in modern web browsers and so does not require the Silverlight plugin, but operates on the same data - so you can work on the same review using either this application, or the main EPPI-Reviewer 4 application.
We are adding more functionality to this version all the time, but at the present time for some features you will need to use the older Silverlight version.

We encourage all users to provide feedback and suggestions by contacting EPPISupport@ucl.ac.uk.
