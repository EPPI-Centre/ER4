Welcome to the EPPI-Reviewer Coding App.

This is an Experimental Application. You should feel free to use it, provided that you are well aware that it is Work in Progress and that it might contain bugs and/or annoyances.

We encourage all users to provide feedback and suggestions by contacting EPPISupport@ucl.ac.uk.

NOTE: this application provides most of the functionalities available through the Coding Only user interface of EPPI-Reviewer 4. It is NOT a replacement for EPPI-Reviewer 4 and cannot be used without first setting up adequate Coding Assignments therein.
