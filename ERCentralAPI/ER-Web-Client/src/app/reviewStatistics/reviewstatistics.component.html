Advanced Bulk Delete

This dialog allows you to delete coding in bulk, on a per-person, per coding tool basis.
Please be aware that this operation destroys data in bulk so please use with care.
If in doubt, please contact us at EPPISupport@ucl.ac.uk.
It's necessary to collect "preview" data because the headline per-person figures do not provide the full picture.
Specifically, items with incomplete coding, which however have coding completed by someone else, do not participate in the "coding progress" per-person "incomplete" counts. Moreover, the "coding progress" figures ignore duplicates and deleted items, but, if present, their coding will be deleted too!
Thus, before you can "bulk delete" coding, we will collect and show the complete figures below.
Please note that for practical reasons we do not count how many "Full Text Selections" and "Outcome Measures" will be deleted along with all the other coding data!
